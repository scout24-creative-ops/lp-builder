# Agent Upload Manifest

- `agent-upload/` ist ein abgeleiteter Bereich und nicht Source of Truth.
- Es gibt aktuell noch kein produktives Bundle in diesem Ordner.
- Pflichtquellen fuer ein spaeteres Bundle:
  - `agent/systemprompt.md`
  - `agent/guardrails.md`
  - `agent/tone-of-voice.md`
  - `knowledge/component-library.html`
  - `knowledge/icon-library.html`
- Optional bzw. erst nach Klaerung:
  - `runtime/core/*`
  - `runtime/integrations/*`
  - `runtime/legacy/*`
- Runtime-Dateien duerfen nicht pauschal oder automatisch in ein Bundle aufgenommen werden, bevor ihr konkreter Bundle-Zweck geklaert ist.
